# HerosEvil
1. 初次使用请使用--help参数获取使用说明
`HerosEvil.exe --help`
2. 使用`server`命令获取登陆区服代码
4. 使用`run`命令运行项目
5. 注意填写账号、密码、区服参数
6. 需在自然状态下运行脚本,无法在打桩、双修、钓鱼、修炼等状态下进行。
# Example：
`HerosEvil.exe server`
`HerosEvil.exe run -u test -p 123456 -s h6`
